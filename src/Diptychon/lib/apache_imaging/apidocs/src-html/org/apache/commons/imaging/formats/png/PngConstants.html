117         * Background colour Solid background colour to be used when presenting the
118         * image if no better option is available. Gamma and chromaticity Gamma
119         * characteristic of the image with respect to the desired output intensity,
120         * and chromaticity characteristics of the RGB values used in the image. ICC
121         * profile Description of the colour space (in the form of an International
122         * Color Consortium (ICC) profile) to which the samples in the image
123         * conform. Image histogram Estimates of how frequently the image uses each
124         * palette entry. Physical pixel dimensions Intended pixel size and aspect
125         * ratio to be used in presenting the PNG image. Significant bits The number
126         * of bits that are significant in the samples. sRGB colour space A
127         * rendering intent (as defined by the International Color Consortium) and
128         * an indication that the image samples conform to this colour space.
129         * Suggested palette A reduced palette that may be used when the display
130         * device is not capable of displaying the full range of colours in the
131         * image. Textual data Textual information (which may be compressed)
132         * associated with the image. Time The time when the PNG image was last
133         * modified. Transparency Alpha information that allows the reference image
134         * to be reconstructed when the alpha channel is not retained in the PNG
135         * image.
